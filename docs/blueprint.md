# **App Name**: US Roofing Rebuild

## Core Features:

- Homepage Replication: Replicate the exact layout and content of the US Roofing homepage, including sections for services, testimonials, and contact information.
- Services Page Cloning: Clone the individual service pages (e.g., roof repair, installation) with all existing content, images, and formatting.
- Contact Form Integration: Implement a contact form that mirrors the existing form, capturing user input for name, email, phone, and message.
- Image Gallery Recreation: Rebuild the image gallery, including all existing images and the same display format.
- Styling with Tailwind CSS: Use Tailwind CSS to ensure accurate styling, responsiveness, and visual consistency with the original website. Create and save a custom tailwind theme file with these specs.
- SEO metadata generation: Generate the page's metadata by crawling the target website using a tool.
- Chatbot Replication: Replicate the personalized chatbot including messages and style, that is present on the target website.

## Style Guidelines:

- Primary color: Deep blue (#003366) to match US Roofing's branding and convey trust and reliability.
- Background color: Light gray (#F2F2F2) for a clean and professional backdrop.
- Accent color: Bright orange (#FF9933) for calls-to-action and highlights, providing a visual contrast.
- Font: 'Inter', a grotesque-style sans-serif font that will be used for both headlines and body text to create a modern, clean, and highly readable user interface.
- Use a responsive grid layout with Tailwind CSS to adapt to different screen sizes, ensuring a consistent experience across devices.
